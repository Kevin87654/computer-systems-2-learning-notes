# 计算机系统（2）课程学习记录

这是我在深圳大学学习“计算机系统（2）”课程期间完成的实验记录，主要内容来自课程实验报告。

课程实验大多在 Linux 虚拟机中完成，代码、命令执行过程和结果以截图形式保存在报告中。因此，本仓库以脱敏后的实验报告为主要归档形式。

- 报告采用 DOCX 格式，请下载后使用 Microsoft Word、WPS Office 或 LibreOffice 查看。
- 姓名、学号、班级以及文档作者元数据已做脱敏处理。
- 报告中的终端用户名、内存地址、Cookie 和测试结果仅用于说明当时的本地实验环境，不是线上账号凭据。
- 本仓库只用于个人学习记录与技术交流，不包含课程课件、测验答案、历年试题、实验二进制文件或构建缓存。
- 部分实验涉及逆向分析和缓冲区溢出，仅应在授权的教学环境中复现。

## 学习收获

通过这些实验，我进一步理解了程序从源代码到机器级执行的过程，包括：

- Linux 环境下的编译、运行与调试流程；
- 整数、浮点数和位级数据表示；
- 汇编代码、函数调用约定和栈帧结构；
- 缓冲区溢出的基本原理及相关安全机制；
- Cache 访问模式对程序性能的影响。
